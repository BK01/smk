## Type Property
`"type"`: `String` *(Required)*

The name of the type of this tool.
Identifies the tool in the configuration, this is required.

Must be one of these values:
[`"about"`](about),
[`"basemaps"`](basemaps),
[`"bespoke"`](bespoke),
[`"coordinate"`](coordinate),
[`"directions"`](directions),
[`"identify"`](identify),
[`"layers"`](layers),
[`"legend"`](legend),
[`"list-menu"`](list-menu),
[`"location"`](location),
[`"markup"`](markup),
[`"measure"`](measure),
[`"minimap"`](minimap),
[`"pan"`](pan),
[`"query"`](query),
[`"scale"`](scale),
[`"search"`](search),
[`"select"`](select),
[`"shortcut-menu"`](shortcut-menu),
[`"toolbar"`](toolbar),
[`"version"`](version),
or
[`"zoom"`](zoom).
